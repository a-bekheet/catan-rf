# Roadmap

## Milestone 1: Engine + Board (current)
- Deterministic board generation with 54 vertices and 72 edges.
- Initial game state and core data structures.

## Milestone 2: Full Rules Engine
- Legal move generation.
- Resource distribution, robber, trading, dev cards.
- Victory conditions and terminal state detection.

## Milestone 3: Environment API
- Gym-style interface with fixed observation schema.
- Action encoding/decoding for agents.
- Baseline random and heuristic agents.

## Milestone 4: RL Scaffolding
- Self-play harness and replay buffer.
- Evaluation pipeline + checkpoints.
- Training scripts with reproducibility logs.

## Milestone 5: Performance + Research
- Policy and value networks.
- Multi-agent training regimes.
- Ablations and reporting.
